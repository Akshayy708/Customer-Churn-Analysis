# Customer-Churn-Analysis

## Overview

This project focuses on analyzing customer churn data to identify patterns and factors influencing customer retention. The analysis was performed using Python with libraries such as Pandas, Matplotlib, and Seaborn.

## Objectives

* Clean and preprocess the dataset
* Perform Exploratory Data Analysis (EDA)
* Identify customer behaviors linked to churn
* Visualize trends and patterns using charts and plots

## Key Insights

* Around 26.5% of customers have churned.
* Senior citizens show a comparatively higher churn rate.
* Customers with month-to-month contracts are more likely to churn.
* Customers using Electronic Check as a payment method have higher churn probability.
* Customers with shorter tenure tend to churn more frequently.
* Services like OnlineSecurity, TechSupport, and OnlineBackup are associated with better customer retention.

## Technologies Used

## Conclusion

The analysis highlights several customer behaviors and service patterns that strongly influence churn. These insights can help businesses improve customer retention strategies and reduce churn rates.
